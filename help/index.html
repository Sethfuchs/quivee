A) HTML Structure - top

This theme is a responsive layout based on Bootstrap. Here below an example of the homepage

If you would like to edit the color check the file css/colors.css you will find four pre-select color schemes.
To add another color scheme add CLASS to the tag body as follow

Explame:
<body class="page-turquoise">

Also check the file js/custom.js and change the color at the line 35
barColor: '#72c1b0'

C) JavaScript - top

This theme imports five Javascript files.

  1. jquery.js
  2. bootstrap.min.js
  3. jquery.scrollTo.min.js
  4. jquery.easing.1.3.js
  5. html5shiv.js
  6. ilightbox.packed.js
  7. jquery.easy-pie-chart.js
  8. jquery.isotope.min.js
  9. jquery.mousewhell.js
  10. jquery.parallax-1.1.3.js
  11. waypoints.min.js
  12. jPreloader.js
  13. custom.js
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. Plugins can be included individually (though some have required dependencies), or all at once. Both bootstrap.js and bootstrap.min.jscontain all plugins in a single file.
  3. A jQuery plugin that enable smooth page scrolling working with anchors
  4. A jQuery plugin from GSGD to give advanced easing options. 
  5. Html5shiv code adds new HTML5 elements (which is simple code), but also supports printing HTML5 elements and includes the default styles for HTML5 elements, like block on article and section.
  6. iLightBox allows you to easily create the most beautiful responsive overlay windows using the jQuery JavaScript library.
  7. Easy pie chart is a jQuery plugin that uses the canvas element to render simple pie charts for single values. These charts are highly customizable, very easy to implement, scale to the resolution of the display of the client to provide sharp charts even on retina displays, and use requestAnimationFrame for smooth animations on modern devices.
  8. Isotope: An exquisite jQuery plugin for magical layouts
  9. A jQuery plugin that adds cross-browser mouse wheel support.
  10. jQuery Parallax is a plugin that provides parallax scrolling effects to any scrolling element.
  11. Waypoints is a jQuery plugin that makes it easy to execute a function whenever you scroll to an element.
  12. jPreLoader can create a preloading screen for your website before all your images (including the images in CSS) are fully loaded.
  13. All custom js code
